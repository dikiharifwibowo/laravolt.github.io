Ada Apa Aja Sih?

Core

Authentication

Login, logout, registrasi (manual atau via media sosial), lupa password, aktivasi akun. Semua sudah tersedia, memanfaatkan fitur autentikasi bawaan Laravel yang sudah teruji disertai beberapa modifikasi.

User & Role Management

Fitur penanganan pengguna yang lengkap dan mudah untuk dimodifikasi, termasuk pengaturan profil pengguna.

Password Management

Pengaturan password yang lebih aman dan canggih, memastikan pengguna tetap aman selama dalam aplikasi.

Application Settings

Admin bisa mengatur konfigurasi aplikasi lewat halaman web, tanpa harus mengubah file konfigurasi secara langsung. Jika terjadi kesalahan, maka "Reset To Default" tetap bisa dilakukan.

Audit Trail & Log Activity

Catat setiap aksi pengguna dan perubahan data yang terjadi. Ketika ada sesuatu yang salah, Anda dengan cepat bisa tahu siapa pelakunya.

Bootstrap & Semantic-UI

Sudah tersedia beberapa template layout yang siap digunakan. Setiap komponen juga selalu disertai dengan view yang tinggal pasang.

CRUD Builder (Aha!)

Membuat fitur create, read, update, delete, searching, sorting dan export secara otomatis berdasar struktur tabel.

Additional

Mural

Mirip facebook comment atau disqus, hanya saja yang ini terintegrasi langsung dengan aplikasi Laravel Anda.

Votee

Widget untuk melakukan like atau dislike.

Star

Menambahkan fitur rating 5 bintang otomatis untuk konten Anda.

Senarai

Mengumpulkan konten dalam sebuah list tertentu. Bisa dipakai untuk fitur 'wishlist', 'bookmark', dan lain-lain.

Avatar

Setiap pengguna yang login sudah langsung punya avatar default tanpa harus upload apapun.

Media Manager

Memantau dan mengatur semua file yang diupload oleh pengguna dari satu halaman.
